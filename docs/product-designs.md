## 产品概述

subtitler 是一个 Rust 实现的高性能字幕处理库，支持 13 种字幕格式的解析、转换、验证和编辑。基于其强大的核心能力，可以构建多个面向不同用户群体的产品和工具。

### 核心能力矩阵

| 能力 | 描述 | 适用场景 |
|------|------|---------|
| **格式转换** | 13种格式互转 | 在线工具、批量处理 |
| **质量检查** | 重叠、时长、CPS验证 | 专业质检、合规审查 |
| **时间编辑** | 偏移、缩放、帧率转换 | 批量调整、后期制作 |
| **文本规范化** | OCR修复、空白清理、标签处理 | 文本清理、格式标准化 |
| **流式处理** | 大文件流式解析 | 服务器应用、实时处理 |
| **广播级支持** | SCC、EBU STL专业格式 | 广播电视、流媒体平台 |

## 技术优势

### 🚀 性能优势
- **Rust 实现**：编译优化，接近 C/C++ 性能
- **零拷贝设计**：减少内存分配（计划 v2.0）
- **流式处理**：支持 GB 级别文件处理
- **并发安全**：原生支持多线程

### 🎯 功能优势
- **格式完整**：13 种主流格式全覆盖
- **专业级**：支持广播级格式（SCC、EBU STL）
- **质量保证**：216 个测试用例，生产级代码质量
- **跨平台**：支持编译为 WASM，浏览器运行

### 📦 技术栈优势
- **单一依赖**：无需外部工具（如 FFmpeg）
- **纯 Rust**：易于集成和部署
- **类型安全**：编译时错误检查
- **文档完善**：API 文档齐全

## 产品设计

### 1. 在线字幕工具平台（Web App）

**产品定位：** 面向普通用户和轻度专业用户的在线字幕处理平台

#### 核心功能

##### 字幕格式转换器
- 支持 13 种格式互转
- 拖拽上传，一键转换
- 批量文件处理（最多 50 个文件）
- 格式兼容性检查

##### 字幕质量检查器
- 自动检测：重叠、负时长、零时长、时间倒序
- 扩展检查：文字过长、CPS 过高、间隔过大
- 可视化报告：时间轴图表、问题高亮
- 修复建议：一键修复常见问题

##### 字幕时间轴调整工具
- 时间偏移：±毫秒调整
- 批量调整：应用到所有字幕
- 帧率转换：23.976 ↔ 25 ↔ 29.97
- 预览功能：实时查看调整效果

##### 字幕文本规范化工具
- OCR 错误修复：0→o, l→1, O→0
- 空白字符清理：多余空格、换行
- 引号标准化：智能引号 → ASCII
- 听障标签移除：[LAUGHS], (APPLAUSE)

##### 在线字幕编辑器
- 时间轴可视化编辑
- 文本实时编辑
- 预览播放（配合视频）
- 多格式导出

#### 技术方案

```
Frontend:
  - React/Next.js + TypeScript
  - Tailwind CSS + shadcn/ui
  - WASM: subtitler compiled to WebAssembly
  - 状态管理: Zustand
  - 文件处理: Browser File API

Backend:
  - Rust: Axum/Actix-web
  - subtitler library
  - 文件存储: Cloudflare R2 / AWS S3
  - 任务队列: Redis + BullMQ

部署:
  - 前端: Vercel / Cloudflare Pages
  - 后端: Railway / Render
  - CDN: Cloudflare
```

#### 用户价值

**普通用户：**
- 无需安装软件，跨平台使用
- 快速处理字幕文件
- 简单易用的界面

**专业用户：**
- 专业级质量检查
- 精确时间轴调整
- 批量处理功能

#### 商业模式

- **免费版**：单个文件处理，基础功能
- **专业版**：批量处理，高级功能，$9.99/月
- **企业版**：API 访问，无限制处理，$49.99/月

#### 实现难度：⭐⭐⭐（中等）

### 6. 字幕API服务（B2B SaaS）

##### RESTful API

**格式转换 API**
```
POST /api/v1/convert
{
  "source_format": "srt",
  "target_format": "vtt",
  "content": "..."
}
```

**质量检查 API**
```
POST /api/v1/validate
{
  "format": "srt",
  "content": "...",
  "options": {
    "basic": true,
    "extended": {
      "max_chars": 42,
      "max_cps": 25.0
    }
  }
}
```

**时间调整 API**
```
POST /api/v1/shift
{
  "format": "srt",
  "content": "...",
  "offset_ms": 1000
}
```

**文本规范化 API**
```
POST /api/v1/normalize
{
  "format": "srt",
  "content": "...",
  "operations": ["fix_ocr", "strip_hi", "normalize_quotes"]
}
```
